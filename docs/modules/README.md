# 模块地图

| 模块域 | 页面目录 | 功能与文档重点 |
| --- | --- | --- |
| 认证与登录 | `auth/`、`login/` | 登录、令牌恢复、验证码和登录跳转；见 [登录流程](../login-flow.md)。 |
| 系统管理 | `system/` | 用户、角色、菜单、资源、租户、字典、配置、岗位等管理能力。 |
| 系统运维 | `sysadmin/`、`security/` | 审计、在线用户、验证码场景、通知、限流、使用统计。 |
| 网关路由 | `system/gateway-route/` | 路由查看、编辑、校验与发布；必须与服务端权限和 Nacos 配置流程一致。 |
| 开发工具 | `development/`、`codegen/` | API 文档和开发辅助页面。 |
| 个人中心 | `profile/` | 个人资料与消息入口。 |

每个新页面/页面域应增加独立 Markdown，至少包含：介绍、功能、后端 API、权限、交互流程、验证方法和规划。
